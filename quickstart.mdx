## One-Click Download

The fastest way to get started. OpenSwarm ships as a standalone macOS `.dmg` — Python is bundled, so there's nothing extra to install.

<Steps>
  <Step title="Download the app">
    Go to [openswarm.info](https://www.openswarm.info) and click **Download for macOS**.

    Open the `.dmg`, drag **OpenSwarm** into your Applications folder, and launch it.
  </Step>
  <Step title="Spawn your first agent">
    You'll land on an **infinite canvas** — your mission control. Click **+ New Agent** to create your first agent card.

    Give it a task. Watch it go.
  </Step>
  <Step title="Explore the dashboard">
    - **Approvals** — All agent tool calls surface in one place. Approve, deny, or edit before execution.
    - **Cost tracker** — Real-time USD spend per agent, always visible.
    - **Agent modes** — Switch between Agent, Ask, Plan, App Builder, and Skill Builder to scope what each agent can do.
  </Step>
</Steps>

<Note>
  OpenSwarm requires macOS. Linux and Windows support is on the roadmap.
</Note>

## Clone the Repo

For developers who want to run from source, contribute, or build on top of OpenSwarm.

### Prerequisites

- **Node.js** 18+ and **npm**
- **Python** 3.11+ (or use the bundled runtime)
- An **Anthropic API key**

### Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/openswarm/openswarm.git
    cd openswarm
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    # Frontend
    npm install

    # Backend
    cd backend && pip install -r requirements.txt && cd ..
    ```
  </Step>
  <Step title="Set your API key">
    ```bash
    export ANTHROPIC_API_KEY=sk-ant-...
    ```

    Or add it to a `.env` file at the project root.
  </Step>
  <Step title="Run the app">
    ```bash
    npm run dev
    ```

    This starts both the FastAPI backend and the Electron frontend together.
  </Step>
</Steps>
