---
title: "Navigation"
description: "Zoom, pan, and navigate the infinite dashboard canvas."
---

The dashboard canvas is infinite and freely navigable. You can zoom in, zoom out, and pan in any direction using your mouse, trackpad, or keyboard.

---

## Trackpad

| Gesture | Action |
| --- | --- |
| **Two-finger swipe** | Pan in any direction |
| **Pinch** | Zoom in/out, centered on cursor |

<Note>
  Pinch-to-zoom sensitivity can be adjusted in **Settings → Zoom Sensitivity** 
</Note>

---

## Mouse

| Action | Result |
| --- | --- |
| **Middle click & drag** | Pan |
| **Right-click drag** | Pan |
| **⌘/Ctrl \+ left-click drag** | Pan |
| **Space \+ left-click drag** | Pan |
| **Scroll-wheel** | Scroll vertically |
| **Shift \+ Scroll-wheel** | Scroll horizontally |
| **⌘/Ctrl \+ Scroll wheel** | Zoom |

---

## Keyboard

| Shortcut | Action |
| --- | --- |
| **⌘ \+** | Zoom in (centered on viewport) |
| **⌘ −** | Zoom out (centered on viewport) |
| **⌘ 0** | Reset zoom to 100% and pan to origin |

---

## Canvas Controls widget

A small control bar appears in the bottom-right corner of the canvas:

| Control | Description |
| --- | --- |
| **−** | Zoom out |
| **\+** | Zoom in |
| **⊡ Fit to view** | Fit all cards into the viewport with padding |
| **✦ Tidy layout** | Auto-arrange all cards into a clean grid |

**Fit to view** — Automatically zooms and pans so all cards are visible. Also runs on first load.

**Tidy layout** — Rearranges all cards into a structured grid. Useful after heavy agent activity.
