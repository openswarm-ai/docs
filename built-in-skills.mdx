## Overview

OpenSwarm ships with two types of built-in capabilities:

1. **Pre-installed Skills** — Markdown instruction files that teach agents how to handle common tasks (document creation, frontend design, etc.)
2. **MCP Tool Integrations** — Connected services that give agents the ability to interact with external platforms (Gmail, Google Drive, Reddit, etc.)

Both are available out of the box. Skills guide *how* the agent works; MCP tools give it *what* it can access.

---

## Pre-installed Skills

These skills come installed in `~/.claude/skills/` and are automatically available to agents:

| Skill | What It Does |
|-------|-------------|
| **Skill Creator** | Meta-skill for creating, testing, and optimizing new skills |
| **Frontend Design** | Guides creation of distinctive, production-grade UI — avoids generic "AI slop" aesthetics |
| **DOCX** | Creates, reads, edits, and manipulates Word documents with professional formatting |
| **PDF** | Handles all PDF operations — reading, merging, splitting, rotating, watermarking, OCR |
| **Themed UI Design** | Builds theme-compliant UI components matching OpenSwarm's design token system |

<Tip>
You can view, edit, or delete any pre-installed skill from the **Skills** page. They work exactly like custom skills — they're just already there when you start.
</Tip>

---

## Skill Registry

Beyond the pre-installed skills, OpenSwarm includes a browsable **Skill Registry** sourced from the [Anthropic Skills repository](https://github.com/anthropics/skills). The registry is organized by category and refreshes automatically.

From the Skills page, you can:
- **Browse** community skills in the sidebar, organized by category
- **Search** by name or description
- **Preview** — click any registry skill to see its full content rendered as Markdown
- **Install** — one click copies the skill to your local `~/.claude/skills/`
- **Edit & Install** — customize a registry skill before installing

## MCP Tool Integrations

MCP tools are separate from skills — they're external service connections powered by the **Model Context Protocol**. These give agents the ability to *do things* in the real world: send emails, read spreadsheets, browse the web, post to social media.

### Google Workspace

Connect your Google account and give agents access to your full Workspace suite.

| Tool | What It Enables |
|------|----------------|
| **Gmail** | Read, search, draft, send, and reply to emails |
| **Google Calendar** | Create, read, update, and delete calendar events |
| **Google Drive** | Search, upload, read, and manage files and folders |
| **Google Docs** | Create and edit documents, append or insert content |
| **Google Sheets** | Read and write spreadsheet data, manage sheets |
| **Google Slides** | Create presentations and manage slides |

### MCP Server Registry

OpenSwarm also includes a full **MCP Server Registry** — a catalog of community and Google MCP servers, sourced from the [official MCP registry](https://registry.modelcontextprotocol.io) and Google's MCP catalog. This registry is separate from the Skills registry:

- **Browse and search** hundreds of MCP servers
- **Sort by GitHub stars** to find popular integrations
- **Filter by source** — community servers vs. Google-maintained servers
- **View details** — descriptions, repository links, environment variables needed

### Browser

The browser tool lets agents interact with the web visually — not just fetch URLs, but actually *use* websites like a human would.

- Navigate to any URL
- Click buttons, fill forms, interact with dynamic content
- Extract information from pages
- Handle multi-step interactions

### Reddit

Agents can read and research content across Reddit:

- Browse subreddits by sort order (hot, new, top, rising)
- Search posts across all of Reddit or specific communities
- Fetch full post content and comment threads
- Analyze user activity and posting history

### Twitter / X

Agents can read, post, and interact on Twitter/X:

- Post tweets and threads
- Search tweets and users
- Read timelines, mentions, and replies
- Like, retweet, bookmark, and follow
